# Act 3: Analysis & Reporting

## Context

Third stage of the CASBO 2026 demo. Tagged survey data is joined with HR
demographics, enabling cross-tabulation and pattern discovery.

## Data

- `data/survey_full_dataset.json` — pre-joined survey + HR data (500 records, 20 fields)

## Goal

Demonstrate AI-driven analysis that surfaces patterns in the data:
- Obvious patterns (visible from survey data alone)
- Intermediate patterns (require cross-tabulation or theme analysis)
- Transition toward the hidden patterns that require deeper investigation

## What to Show

- How to prompt AI to explore data systematically
- Chart/visualization generation
- Report drafting with proper caveats and sourcing
- Claude Code takes more of the spotlight here — live pattern discovery.
